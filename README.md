# 단웅이의 헬스케어

> 단국대학교 소프트웨어개론 팀 프로젝트 | 2026년 1학기

## 프로젝트 개요

운동(헬스)과 식습관 교정에 관심 있는 사람들을 위하여, 운동 계획과 실행의 불편함을 해결하는 개인 맞춤형 피드백 소프트웨어.

## 주요 기능

- 운동 수행 중 실시간 자세 확인
- 자세 오류 감지 및 즉각 피드백 제공
- 관절 부담 동작 발생 시 경고
- 헬스장 환경에 따른 대체 운동 제안
- 컨디션에 따른 운동 강도 조정 추천
